# Task 1 - To-Do List Application

## Description
A simple Python To-Do List application that allows users to:
- Add new tasks
- View all tasks
- Exit the application

## Features
- Menu-driven program
- Easy to use
- Beginner-friendly Python project

## Technologies Used
- Python 3

## How to Run
1. Open the terminal.
2. Run:
   python "To Do list.py"

## Output
The program displays a menu to add and view tasks.
