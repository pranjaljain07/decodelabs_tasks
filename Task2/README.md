# Task 2 - Expense Tracker

## Description
A simple Python Expense Tracker that allows users to:
- Add expenses
- View all expenses
- Calculate total expenses
- Exit the application

## Features
- Menu-driven program
- Beginner-friendly
- Easy to use

## Technologies Used
- Python 3

## How to Run
1. Open the terminal.
2. Run:
   python Expense_tracker.py

## Output
The program allows users to manage daily expenses.
