# Task 3 - Random Password Generator

## Description

A simple Python application that generates strong and secure random passwords based on the user's desired password length. The generated password contains uppercase letters, lowercase letters, digits, and special characters to improve security.

## Features

- Generate random passwords
- User-defined password length
- Includes uppercase letters
- Includes lowercase letters
- Includes numbers
- Includes special characters
- Strong and secure password generation
- Easy to use

## Technologies Used

- Python 3
- secrets module
- string module

## How to Run

1. Open the project in VS Code.
2. Run the `Password_Generator.py` file.
3. Enter the desired password length.
4. The program will generate a secure random password.
